## 🏗 Core Systems

### 🔹 Cluster Builder (Multi-Cloud Orchestration Engine)

A distributed infrastructure orchestration platform for automated provisioning and lifecycle management of Kubernetes (K3s) clusters across AWS, OpenStack, and edge devices.

**Key capabilities:**
- Automated infrastructure provisioning using OpenTofu
- Multi-cloud cluster orchestration (AWS, OpenStack, edge)
- Dynamic node lifecycle management
- PostgreSQL-backed state management
- SSH-based remote provisioning
- Modular orchestration engine design

---

### 🔹 K3s Client (Cluster Control Interface)

A Python-based client interface for interacting with distributed Kubernetes clusters managed by Cluster Builder.

**Key capabilities:**
- Programmatic cluster management
- Node-level operations across environments
- API abstraction for orchestration systems
- Multi-cloud cluster interaction layer
- Extensible control interface design
