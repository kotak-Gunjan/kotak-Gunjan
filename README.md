## 🏗 Core Systems

### 🔹 Cluster Builder — Multi-Cloud Orchestration Engine

A distributed infrastructure orchestration platform for automated provisioning and lifecycle management of Kubernetes (K3s) clusters across AWS, OpenStack, and edge environments.

**Core capabilities:**
- Infrastructure provisioning using OpenTofu
- Multi-cloud orchestration (AWS, OpenStack, edge devices)
- Kubernetes (K3s) cluster lifecycle automation
- Dynamic node management (scale up/down, replacement)
- PostgreSQL-backed state management
- SSH-based provisioning and remote execution
- Modular orchestration engine architecture

---

### 🔹 K3s Client — Cluster Control Interface

A Python-based client interface for interacting with distributed Kubernetes clusters managed by Cluster Builder.

**Core capabilities:**
- Programmatic cluster lifecycle interaction
- Node-level management across environments
- API abstraction for orchestration workflows
- Multi-cloud cluster control interface
- Extensible integration layer for automation systems
